- 🎨 Modern and responsive UI.
- ⚡ Smooth animations and transitions.

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js
- npm
- Git

### Installation

Clone the repository:

```bash
git clone https://github.com/RavendraPatel09/New-Portfolio.git
```

Move into the project directory:

```bash
cd New-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
